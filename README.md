# The Boom Code App

A **Power Apps Code App** providing full CRUD for 8 Dataverse tables: Accounts, Contacts, Action Items, HVAs, Ideas, Impacts, Meeting Summaries, Projects.

## Tech Stack

React 19 · Vite 7 · TypeScript 5.9 · @microsoft/power-apps SDK · shadcn/ui · Tailwind CSS 4 · React Query · react-router-dom

## Quick Start

```bash
cd "The Boom Code App"
npm install
npm run build
pac code push
```
